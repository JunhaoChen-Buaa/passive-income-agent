# Security Policy

## API Key

被动收益 Agent 的 API Key 只应保存在本地环境或本地数据库中。

请不要提交：

- `.env`
- `backend/data/`
- 服务器部署密钥
- 私人截图中出现的密钥、账号或资产信息

`.env.example` 只保留空示例，不应包含真实密钥。

## Reporting

如果你发现安全问题，请不要在公开 issue 中贴出真实密钥、日志或个人资产信息。请先用脱敏信息描述问题。

## Scope

本项目不接券商、不自动下单、不保存券商账号密码。任何接入真实交易账户的功能都不属于当前 MVP 范围。
